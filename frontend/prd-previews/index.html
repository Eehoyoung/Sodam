SODAM PRD HTML OUTPUTS

PRD 기준 서비스 시뮬레이션 산출물

각 파일은 브라우저에서 바로 열 수 있는 정적 HTML입니다. 역할별 PRD를 빠르게 리뷰하고, 핵심 상태와 액션을 눌러볼 수 있도록 나눴습니다.

ALL

통합 서비스 앱 미리보기

비회원, 사장님, 직원, 운영자 서비스를 앱 폰 목업 안에서 전환하며 전체 흐름을 확인합니다.

 PRD-GUEST

비회원·온보딩

Splash, 역할 선택, 가입/로그인, 약관, 기본정보, 역할별 홈 진입.

 PRD-OWNER

사장님 서비스

홈 대시보드, 출퇴근 보드, 급여 정산, 매장·직원, 정보허브.

 PRD-EMPLOYEE

직원 서비스

빠른 출퇴근, NFC/GPS, 근무 캘린더, 급여 명세, 매장 가입, 알림.

 PRD-ADMIN

운영자 백오피스

KPI, 사용자 관리, 결제 운영, CMS, 푸시/시스템.

 AUTH FLOW

신규 사용자 인증/온보딩 상세

이전 작업에서 만든 역할 선택 → 가입 → 로그인 → 약관 → 기본정보 → 역할별 홈 상세 미리보기입니다.
