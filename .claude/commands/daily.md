# /daily — 데일리 스탠드업

CEO/CTO/CFO 1인 통합 관점에서 오늘 무엇을 할지 30초 안에 결정한다.

## 출력 양식

```
[어제 한 것]
- ...

[오늘 막힌 것 / 인간 승인 대기]
- ...

[오늘 1순위 액션 1개]
→ (구체적 작업, 예상 소요)

[KPI 한 줄]
- 가입 사업장 / 유료 / MRR / 출시까지 D-XX
```

## 규칙

- 1순위 액션은 **반드시 1개만**. 2개 이상이면 우선순위 명시.
- 추정 소요는 오늘 안에 끝낼 수 있는 크기로.
- 인간 승인이 필요한 작업이면 그렇다고 명시.
