### 4-2. 핵심 기능 추가 (대분류)

#### A. 결제·구독 풀스택 (완전 신규)
- 4 플랜 카탈로그 enum (`PlanType`)
- 5 상태 머신 (`SubscriptionStatus`)
- 빌링키 발급 → 정기 청구 → 웹훅 → 해지 전체 흐름
- HMAC-SHA256 웹훅 서명 검증
- 매일 03:00 자동 청구 스케줄러 (KST)
- 결제 실패 누적 3회 → EXPIRED 자동 전이

#### B. 푸시 알림 풀스택 (완전 신규)
- 디바이스 토큰 1:N 관리
- 6종 도메인 알림 (출근/퇴근/누락/급여/결제 성공·실패)
- Mock/Live 토글 (FCM Firebase Admin SDK)
- 15분 주기 누락 감지 스케줄러

#### C. dev 프로필 (외부 의존성 0 부팅) — 완전 신규
- H2 in-memory + MySQL 호환 모드
- Redis 없이 `ConcurrentMapCacheManager`
- `TokenStore` 인터페이스 + `InMemoryTokenStore`
- 부팅 시 데모 데이터(사장·직원·매장) 자동 생성

#### D. 디자인 시스템 통일 (완전 신규)

- 브랜드 컬러 `#FF6B35` 확정
- 5 공용 컴포넌트 토큰화

- Rate Limit (Bucket4j 메모리 백엔드)
