# PRD-EMPLOYEE — 직원(EMPLOYEE) 역할 상세 명세

> 알바생·직원이 소담 앱에서 하는 모든 일을 빠짐없이 명세.
> **차별점**: 직원이 본인의 시급·근무시간·급여 명세를 직접 확인 → 사장-직원 분쟁 감소.

**버전**: 1.0
**작성일**: 2026-05-19
**페르소나**: "지훈(24세, 학생, 카페+학원 2개 매장)" — PRD §3.2 참고

---

## 0. 직원이 이 앱으로 해결하는 5대 작업

| # | JTBD | 측정 |
|---|---|---|
| 1 | "출근/퇴근 빨리 찍고 싶다" | 출근 평균 소요 ≤ 5초 |
| 2 | "내가 일한 만큼 정확히 받았는지 확인" | 명세서 열람률 ≥ 80% |
| 3 | "2개 매장 근무, 한 화면에서 보고 싶다" | 멀티 매장 화면 사용률 ≥ 30% |
| 4 | "출근 누락/실수 알고 빨리 정정" | 누락 푸시 24h 내 정정 ≥ 70% |
| 5 | "알바 친구한테 추천하기 쉽게" | 친구 추천 클릭률 ≥ 5% |

## 1. 정보 구조 (IA) — 직원 모드 네비게이션

```
[하단 탭]
├ 출퇴근 (오늘의 매장 + 빠른 출근)
├ 캘린더 (월간 근무 기록)
├ 급여 (월별 명세서)
├ 매장 (소속 매장 목록)
└ 더보기 (내정보·정보허브·Q&A·설정)
```

## 2. 화면별 명세

### E-001 · 출퇴근 홈 (Employee Home)

**상단**: 오늘 인사 ("좋은 아침이에요, 지훈님 ☀️"), 현재 시각.

**중앙 — 큰 동그라미 버튼**:
- 미출근 상태 → 🟢 **"출근하기"** (탭 → S-NFC 또는 S-GPS)
- 근무 중 → 🟠 **"근무 중 03:24"** (탭 → "퇴근하기")
- 퇴근 완료 → 🔵 **"오늘 근무 완료 8시간 12분"**

**하단**:
- 매장 셀렉터 (소속 매장 여러 곳일 때)
- 오늘 시급 표시 (예: "오늘 시급 12,000원")
- 빠른 액션: "이번 달 급여 보기" / "출근 기록 보기"

**엣지케이스**:
- GPS 권한 거부 → "사장님께 수동 등록 요청하기" CTA
- NFC 미지원 단말 → GPS-only 자동 폴백
- 운영시간 외 → "지금은 운영시간이 아니에요. 그래도 출근하시겠어요?" 경고

---

### E-002 · NFC 출근 (NFC Check-in)

**플로우**:
1. 큰 가이드 일러스트: "매장 NFC 스티커에 폰을 가져가 주세요"
2. NFC 감지 → 진동 + "✅ 인증 완료"
3. GPS 자동 확인 (백그라운드)
4. 출근 완료 모달 (오늘 시작 시간, 적용 시급)

**엣지케이스**:
- NFC 태그 ID 불일치 → "다른 매장의 스티커예요" 안내
- GPS 범위 밖 → "매장 근처에서 다시 시도해 주세요 (현재 {거리}m)"
- 네트워크 끊김 → 로컬 큐에 저장, 복구 시 자동 동기화

---

### E-003 · GPS 출근 (GPS Check-in)

**플로우**:
1. 위치 확인 중 스피너 (최대 3초)
2. 매장 반경 내 → "출근 등록할까요?" 확인 → 출근
3. 반경 밖 → 지도 + 현재 위치 표시 + 거리 안내

**엣지케이스**:
- 위치 정확도 낮음(>50m 오차) → "조금 더 기다려 주세요" + 재시도

---

### E-101 · 근무 캘린더

**상단**: 월간 캘린더 (한 달 그리드).
- 출근일: 🟢 점
- 근무중인 날: 🟠 점
- 결근: 🔴 점
- 휴무: 회색 점
- 누락 의심: ⚠️ 점

**하단**: 선택한 날의 상세 카드:
- 출근/퇴근 시각, 총 근무시간
- 적용 시급, 일급(예상)
- 위치 인증 결과
- 메모 (사장이 추가했을 시)

**액션**:
- 이상 신고 → 사장에게 정정 요청 (사유 입력)

---

### E-201 · 급여 목록

월별 카드 리스트:
- 월/매장명 / 상태(DRAFT/APPROVED/PAID/CANCELLED) / 실수령액
- 탭 → E-202 상세

---

### E-202 · 급여 명세서 상세

**구성**:
1. 매장명 + 정산 기간
2. 근무 시간 분해 (기본/연장/야간/주휴)
3. 시급 적용 내역 (변경 이력 포함)
4. 총 급여 (세전)
5. 세금 3.3% 차감
6. 기타 공제 (있다면)
7. **실수령액** (강조 박스)
8. 사장이 첨부한 메모

**액션**:
- "공유" → PDF 다운로드 (Phase 2) 또는 카카오톡 공유 텍스트
- "이의 신청" → 사장에게 코멘트 전송 (사유 + 증빙)

**엣지케이스**:
- 명세서가 아직 생성 안 됨 → "사장님이 정산 중이에요" 안내
- 부분 환불·재정산 → 변경 이력 타임라인 노출

---

### E-301 · 매장 목록 & 가입

**소속 매장 카드 리스트**:
- 매장명·주소·내 시급·역할
- 활성/비활성 토글 (직원이 직접 비활성화 시 결근 차감 X 안내)

**+ 매장 가입 버튼**:
- 입력: 사장이 공유한 매장 코드 (12자 알파넘) 또는 QR 스캔
- 가입 신청 → 사장 승인 대기 (Optional 흐름) 또는 즉시 가입

---

### E-401 · 내 정보

- 프로필: 이름/이메일/전화/카카오 연동
- 입사일 (매장별)
- 비밀번호 변경
- 본인의 잔여 연차 (Phase 2)
- 본인 출퇴근 CSV 다운로드 (이메일 전송)

---

### E-501 · 알림 센터

- 출근 누락 / 급여 명세 도착 / 매장 공지 / 시스템 업데이트
- 알림별 상세 화면으로 딥링크
- 알림 설정 (방해금지 시간대)

---

### E-601 · 정보 허브 (Employee Edition)

직원이 자주 궁금해 하는 콘텐츠 우선:
- 주휴수당 받는 법
- 4대보험 가입 자격
- 최저시급/연장근로 수당
- 퇴직 시 정산 (사대보험 등)
- Q&A: 직원 ↔ 사장 / 직원 ↔ 운영자

## 5. 마이크로 인터랙션 (디테일)

- 출근 완료 시 햅틱 + 미세 폭죽 애니메이션 (LinearGradient + Reanimated)
- 근무 중 타이머는 1초 단위 카운트업 (배터리 영향 X — RAF 사용 X, JS setInterval 1Hz)
- 급여 숫자는 콤마 자릿수 + 단위 ("원" 후행)
- 빈 상태 일러스트는 다정한 톤 ("아직 매장이 없어요 — 매장 코드를 받으셨나요?")
