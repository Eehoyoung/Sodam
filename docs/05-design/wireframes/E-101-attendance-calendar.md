# E-101 · 근무 캘린더 (월간)

## 목적
직원이 본인의 한 달 근무를 한 화면에서 시각화. 일자 탭 → 상세.

## 토큰
- 캘린더 그리드 7열, 일자 셀 44×44 (터치 영역)
- 점 표시기: 6×6, status 별 색
  - 🟢 `attendanceCheckedIn`
  - 🟠 `warning`
  - 🔴 `error`
  - ⚠️ `warning` + 깜빡임
- 선택일 강조: `radius.lg` + `brandPrimary` 1px border
- 일급 숫자: `weights.semibold`, tabular-nums

## 상태
- 데이터 로딩: 셀 회색 placeholder
- 데이터 0건: "아직 출근 기록이 없어요" 빈 상태 일러스트
- 미래 일자 탭 시도: 비활성 (회색 톤)

## API
- GET `/api/attendance/employee/{id}/monthly?year=...&month=...`
