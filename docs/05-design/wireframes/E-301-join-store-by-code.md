# E-301 · 매장 코드 가입

## 목적
직원 본인이 사장님이 공유한 매장 코드(또는 QR)로 매장에 가입.

## 레이아웃
```
┌─────────────────────────────┐
│ ← [뒤로]    매장 가입           │
│                             │
│ 사장님이 알려주신                │
│ 매장 코드를 입력해 주세요         │  (lg bold)
│                             │
│ ┌─────────────────────────┐ │
│ │  ST1234ABCD             │ │  ← 모노스페이스, 자동 대문자
│ └─────────────────────────┘ │
│                             │
│         또는                 │  (xs textTertiary, divider)
│                             │
│ ┌───────────────────────┐   │
│ │                       │   │
│ │  [QR 스캔]            │   │  ← 카메라 뷰 자리
│ │  카메라로 매장 QR        │   │
│ │  스캔                  │   │
│ │                       │   │
│ └───────────────────────┘   │
│                             │
│ [   매장 가입하기    ]       │  ← Primary lg
│                             │
│ 매장 코드는 어디서 받나요?      │  ← ghost link
└─────────────────────────────┘
```

## 토큰
- 코드 입력 박스: `typography.fontFamily.monospace` 또는 `letterSpacing: 4`
- QR 영역: `radius.xl` + `border` dashed + 안내 텍스트 중앙
- 도움말 링크 → 모달 "사장님께 매장 코드 또는 QR 을 받으세요"

## 상태
- IDLE: 코드 입력 또는 QR 스캔 대기
- 카메라 권한 없음: "카메라 권한 허용" CTA + 시스템 설정 딥링크
- 코드 검증 중: 버튼 loading
- 성공: 햅틱 success + Alert "{매장명} 에 가입되었어요!" → 홈 이동
- 실패: 토스트 "매장 코드가 일치하지 않아요. 다시 확인해 주세요"

## API
- POST `/api/stores/join-by-code` { storeCode } → Store

## 엣지케이스
- 이미 가입된 매장 코드 → 상태 메시지 "이미 가입된 매장이에요" (성공으로 처리)
- 비활성화된 매장 → "삭제된 매장이에요. 사장님께 문의해 주세요"

## 접근성
- 입력 필드 keyboard="default", autoCapitalize="characters"
- QR 영역 accessibilityLabel="매장 QR 코드 스캔"
