# O-901 · 세무사 파트너 콘솔 (Phase 2)

## 목적
환급형 플랜 가입자의 케이스를 세무사가 처리하는 별도 콘솔.
세무사 계정 = 별도 등급 `ROLE_TAX_PARTNER`.

## 레이아웃
```
[탭: 케이스 큐 / 진행중 / 완료 / 수수료]
─────────────────────────────────────────
대기 케이스 (8)
  사장        매장      매출(추정)  접수일
  은영(대표)  소담카페   ₩48M        05-18  [열기]
─────────────────────────────────────────
```

### 케이스 상세
- 사장 기본정보 + 사업자번호
- 업로드된 매출/지출 서류 (S3 ObjectStorage)
- 처리 상태 (PENDING / IN_PROGRESS / DONE / REJECTED)
- 환급 예상액 입력 → 실제 환급액 확정 → 수수료 자동 계산

## API
- `GET /api/admin/tax-cases?status=`
- `PUT /api/admin/tax-cases/{id}/status`
- `POST /api/admin/tax-cases/{id}/result` (환급액 입력)

## 권한
세무사 등급만. 사용자 PII 는 케이스 처리에 필요한 최소만 노출.
