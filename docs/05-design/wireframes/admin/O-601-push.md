# O-601 · 푸시 발송 (마케팅)

## 목적
사용자 세그먼트별 푸시 알림 발송. A/B 테스트 + 결과 리포트.

## 레이아웃
```
[+ 새 캠페인]
─────────────────────────────────────────
캠페인명               세그먼트       상태       발송수  오픈
종소세 환급 안내       유료-비즈니스   완료       2,341   34%
1주년 할인             전체           예약       —       —
─────────────────────────────────────────
```

### 새 캠페인 작성
1. 세그먼트: 전체 / 사장 / 직원 / 무료 / 유료 / 환급형
2. 제목 + 본문 (총 100자 이내 권장)
3. 딥링크 (예: `sodam://subscription`)
4. 발송 시점: 즉시 / 예약
5. A/B 테스트 (Phase 2): 두 가지 카피를 50:50 으로 발송 후 오픈률 비교

### 결과 리포트
- 도달 / 오픈 / 클릭 수
- 시간대별 오픈 분포
- 세그먼트별 비교

## API
- `GET /api/admin/push/campaigns`
- `POST /api/admin/push/campaigns` (즉시/예약)
- `GET /api/admin/push/campaigns/{id}/report`

## 권한
전체 사용자 대상 발송 → 재인증.
