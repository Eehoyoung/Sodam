# O-501 · Q&A 모더레이션

## 목적
사장/직원이 등록한 Q&A 검토·답변·FAQ 변환.

## 레이아웃
```
[필터: 사장 / 직원]  [정렬: 최신 / 답변대기]
─────────────────────────────────────────
🔴 답변대기 (12)
  Q. 주휴수당 자동 계산이 잘못된 것 같아요    | 2시간 전
  Q. 환급형 가입 절차가 궁금합니다           | 어제
─────────────────────────────────────────
✅ 답변 완료
  Q. ...
```

### 답변 작성 (행 클릭)
- 원문 + 답변 에디터
- "FAQ 로 변환" 체크박스 → 같은 답변을 FAQ 컬렉션에 자동 추가
- "부적절 질문 숨기기" (사유 + 신고자 표기)
- 답변 등록 시 질문자에게 푸시 자동 발송

## API
- `GET /api/admin/qna?role=&status=`
- `POST /api/admin/qna/{id}/answer`
- `POST /api/admin/qna/{id}/hide` (사유 필수)
- `POST /api/admin/qna/{id}/promote-to-faq`

## 권한
숨김 처리 → 운영자 ID 로깅 (audit_log).
