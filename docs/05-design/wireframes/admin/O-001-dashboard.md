# O-001 · 운영 대시보드

## 목적
운영자가 매일 켜는 첫 화면. KPI · 미응대 CS · 에러 트렌드 · 결제 실패 한눈에.

## 레이아웃
```
+------------------------------------------+
| 소담 운영자  |  검색  |  알림  |  계정    |
+------------------------------------------+
| 오늘 지표                                 |
| [신규 가입] [유료전환] [이탈] [MRR ↗]    |
+------------------------------------------+
| 가입 트렌드 (최근 7일 라인 차트)           |
+------------------------------------------+
| 미응대 CS  |  Sentry 이슈 Top5  | 결제 실패 |
| 12건       |  • TypeError ...    | 3건       |
| [응대하러] |  [Sentry 열기]      | [확인]    |
+------------------------------------------+
```

## 토큰
- KPI 카드 4개 그리드 (2x2 모바일 / 4열 데스크탑)
- 트렌드 라인: `brandPrimary` solid
- 위젯 카드: `shadow.md`

## API
- `GET /api/admin/kpi/today`
- `GET /api/admin/kpi/trend?days=7`
- `GET /api/admin/cs/unread-count`
- Sentry: 외부 임베드 또는 API
- `GET /api/admin/billing/failed?days=7`

## 권한
`ROLE_ADMIN` 만. IP 화이트리스트 권장.
