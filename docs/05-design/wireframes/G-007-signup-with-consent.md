# G-007 · 이메일 회원가입 + 약관 동의

## 목적
KISA 권고 만 14세 이상 확인 + 약관 동의 4종 + 이메일 가입 단일 화면.

## 레이아웃
```
┌─────────────────────────────┐
│ ← [뒤로]                     │
│                             │
│ 이메일로 가입하기              │  (xxl bold)
│ 1분이면 시작할 수 있어요        │
│                             │
│ ┌─────────────────────────┐ │
│ │ 이름 (실명)               │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 이메일                   │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 비밀번호       [보기]      │ │
│ └─────────────────────────┘ │
│ ▓▓▓▓░░░░ 강도: 보통           │
│                             │
│ ─────  약관 동의  ─────       │
│                             │
│ ☐ 전체 동의                  │  ← bold, 라이트 박스
│ ─────────                   │
│ ☐ 만 14세 이상이에요 (필수)   │
│ ☐ 이용약관 동의 (필수)   [보기]│
│ ☐ 개인정보 처리방침 (필수)[보기]│
│ ☐ 마케팅 정보 수신 (선택)     │
│                             │
│ [    가입하기    ]           │  ← Primary lg, 필수 동의 시 활성
└─────────────────────────────┘
```

## 토큰 사양
- 체크박스: 24×24, `brandPrimary` 활성 색, 라운드 4
- 약관 보기: ghost link, "보기" 텍스트는 `brandPrimary`
- 필수 항목 텍스트: "(필수)" 부분만 `error` 톤
- 풀텍스트 모달: BottomSheet 70% 높이, 닫기는 ↓ 핸들

## 상태
- IDLE: 필수 미동의 → 가입 버튼 비활성
- 필수 3종 모두 체크 + 모든 폼 유효 → 가입 버튼 활성 (햅틱 light)
- 가입 진행 중: 버튼 loading
- 에러: 이메일 중복 시 inline ("이미 사용 중이에요. 로그인하시겠어요?" + 링크)

## API
- POST `/api/join` — body 에 `ageConfirmed`, `termsAgreed`, `privacyAgreed`, `marketingAgreed` 포함

## 엣지케이스
- 약관 풀텍스트 → docs/legal/*.md 를 마크다운 렌더링 또는 정적 텍스트로 표시
- "전체 동의" 토글 → 모든 항목 동시 체크/해제

## 접근성
- 각 체크박스 `accessibilityRole="checkbox"`, state checked 반영
- 풀텍스트 모달은 ScrollView accessibilityRole="document"
