# E-501 · 알림 센터

## 목적
받은 푸시 알림을 시간순 + 카테고리별로 다시 보기. 딥링크 진입.

## 레이아웃
```
┌─────────────────────────────┐
│ ←  알림                      │
├─────────────────────────────┤
│ [전체] 출퇴근  급여  결제  공지 │  ← Filter chips
├─────────────────────────────┤
│ ● 5분 전                     │  (xs warning)
│ 출퇴근 누락 알림              │  (lg semibold)
│ 소담 데모 카페의 퇴근 기록이    │  (sm secondary)
│ 누락된 것 같아요.              │
│ ─────────────────           │
│                             │
│ ○ 어제                      │  (xs textTertiary)
│ 급여 입금 완료                │
│ 5월 급여 ₩1,438,896이         │
│ 정산되었어요.                  │
└─────────────────────────────┘
```

## 토큰
- 읽지 않음: 좌측 `brandPrimary` 6px 점 + 진한 폰트
- 읽음: 회색 점 + 일반 폰트
- 카테고리 칩: 활성=`brandPrimary` bg + 흰 글자, 비활성=`surfaceMuted`

## 상태
- 데이터 0: "받은 알림이 없어요" 일러스트
- 새 알림 도착: 무한 스크롤 상단 자동 갱신
- 클릭 시 데이터 fetch 후 딥링크 이동

## API
- GET `/api/notifications/inbox` (P1 — 현재 미구현)
- POST `/api/notifications/{id}/read`

## 엣지케이스
- 90일 이전 알림은 삭제 (BE 배치)
- 한 번에 50개씩 로드
