# 브랜드 카피 일관성 점검 (Self-audit)

## 톤 헌법
- **다정한 존댓말** ("이용해 주세요", "확인해 주세요")
- **사장님 / ○○님** 호칭 통일 (반말 X)
- **금지어**: 혁신, AI 기반, 스마트, 첨단, 차세대, 솔루션
- **권장어**: "사장님", "월말 30분", "환급", "내 시급", "주휴수당", "한 번에"

## 화면별 카피 권장

### OwnerDashboardScreen
- "안녕하세요 / ○○ 님 👋"
- "오늘 출근 현황"
- "이번 달 누적 급여"
- "모든 직원이 출근했어요 ✅"
- "급여 정산하기", "직원 추가", "위치/반경 설정"
- "이번 달 야간 근무가 지난달 대비 늘었어요. 직원 컨디션 확인해 보세요."

### EmployeeAttendanceHome
- "좋은 아침이에요" / "수고하셨어요" (시간대별)
- "출근하기" (단순 명확)
- "근무 중" → 카운트업
- "오늘 근무 완료" + 시간 표시

### Errors
- "잠시 후 다시 시도해 주세요" (사용자 책임 X)
- "우리 잘못이에요" (5xx)
- "매장 코드와 일치하는 매장을 찾을 수 없어요" (404)

### Success
- "출근이 등록되었어요"
- "급여 명세서를 발급했어요"
- "구독이 시작됐어요"

### Empty
- "아직 ~ 없어요" (비난 톤 X)
- "첫 ~ 을 ~ 해볼까요?" (행동 유도)

### Form
- "이용약관 동의 (필수)" — "(필수)"는 error 톤
- "선택" — 항상 명시
- "8자 이상, 대소문자·숫자·특수문자 포함"

## 영문 단어 검출 (한국어로 통일)
| 발견 | → 권장 |
|---|---|
| "Confirm" | "확인" |
| "Submit" | "제출" / "보내기" |
| "Cancel" | "취소" |
| "Save" | "저장" |
| "Next" | "다음" |
| "Done" | "완료" |
| "Login" | "로그인" |
| "Logout" | "로그아웃" |
| "Sign up" | "회원가입" |

## 한자어 vs 순우리말
- 부정적 한자어 → 순우리말:
  - "사용 불가" → "이용할 수 없어요"
  - "오류 발생" → "오류가 났어요"
  - "삭제 완료" → "삭제했어요"

## 호칭 정책
- 사장 페르소나 → "사장님"
- 직원 페르소나 → "○○ 님" (이름)
- 비회원 → "회원님" 또는 "고객님" 보다 자연스러운 호칭 권장 (예: "소담을 둘러보세요")
