# SODAM 수익화·수익모델 — 확정안 (v2.0)

**개정일**: 2026-06-06 (v1.0 → v2.0)
**의결**: 4인 페르소나 토론 — CFO·CEO(재무·성장) + **소상공인 사장님(수요)·세무사(공급)** + 메인(위원장) 판정
**지위**: PRD §5(수익 모델) 대체·확정. 단위는 **원(₩ 약어·k·M 미사용), 파생 지표는 천원 미만 절사**(가격 끝자리 9는 의도적 마케팅값으로 유지).
**전제**: 약관·면책·정기결제 고지 **법률 문구**, 결제 로직, **세무사 계약·세무사법 유권해석**은 인간/전문가 승인 사항(본 문서는 구조·수치·코드연동만 확정).

---

## 0. v2.0 핵심 변경 (왜 바꿨나)

v1.0(CFO·CEO 합의) 위에 **사장님(수요)·세무사(공급)** 검증을 더한 결과, 두 가지가 **근본적으로 틀렸음**이 드러났다:

1. **환급(COMMISSION) 정률 모델은 폐기 수준 재설계.** 세무사 검증: ① 사장님(원천징수의무자) 종소세는 **구조적으로 환급이 거의 안 남**(간이·면세·결손 = 환급 0원 = 타겟 모집단 공백) ② 세무사 손익분기(건당 100,000원)를 수수료(환급 200,000원의 15% = 30,000원)가 못 채워 **세무사 몫이 음수** ③ "환급액 계산"은 **세무사법 위반 회색지대**(삼쩜삼 고발 전례). → **정률 환급을 버리고 정액 신고 송객 + 고용세액공제 특화 + 부가세·원천세 반복 보조**로 피벗.
2. **직원 수 과금은 사용을 억제한다.** 사장님 검증: 직원당 과금이면 **사장이 직원을 앱에 안 넣고 숨긴다**(특히 단기·주말 알바) → 데이터가 새고 가치 하락 → 이탈. 또 STARTER(3명)→PRO(10명) 사이에 **최대 고객층(직원 4~7명)이 갈 곳이 없다.** → **직원 한도를 넉넉히, 차별화는 기능(명세서·4대보험 신고·멀티매장·세무)으로.**

## 3. 단위경제·BEP (원 단위, 천원 미만 절사)

**블렌디드 ARPU**(STARTER 50%·PRO 35%·PREMIUM 15% + 멀티매장 일부) ≈ **16,000~18,000원/월**.
**그로스마진**: ARPU − 변동비(PG 3% + CS잠식 + SMS/지오코딩) ≈ **15,000원/유료/월 (마진 약 82%)**.

| 지표 | 값 |
|---|---|
| **현금 BEP**(창업자급여 제외, 월 고정비 1,000,000원 ÷ 15,000원) | **유료 약 65명 — M8~9 자립 가능** |
| **완전 BEP**(창업자급여 3,500,000원 포함, 4,500,000원 ÷ 15,000원) | **유료 약 290명 — 18개월 트랙** |
| LTV(13개월, 폐업 churn 반영) | 15,000원 × 13 = **195,000원** |
| CAC | 30,000원(낙관)~50,000원(현실) → LTV/CAC = **3.9~6.5** ✅ |

| 시나리오 | 누적가입 | 전환율(M12) | 유료수 | 구독 MRR |
|---|---|---|---|---|
| 보수 | 1,200 | 3% | 약 80 | **1,500,000원** (출시 합격선) |

## 4. 세무 수익 — 환급 정률 폐기 → 정액 신고·고용공제·반복세무 (v2.0 전면 피벗)

> 세무사 검증 결론: **"환급 15% 정률"은 타겟 모집단이 비고, 세무사 몫이 안 나오며, 세무사법 위반 회색지대 — 폐기.** 대신 아래 3축.

### 4-1. 정액 종소세 신고 송객 패키지 (환급 정률 대체)
- **고객 99,000~149,000원/건**(시장가 100,000~200,000원 하회, 구독자 우대). **환급 0원인 사장도 "신고는 해야 하니" 구매** → 환급 의존 제거.
- **분배**: 세무사 **70,000~100,000원**(손익분기 충족 → 파트너 인센티브 해결), 소담 **30,000~49,000원**(데이터 정리·증빙 자동화·송객 SW 이용료).
- **법적 포지셔닝**: 소담은 **정리된 데이터 전달 + 송객(소프트웨어)만**. 신고·계산·상담·**환급액 확정은 100% 세무사**. 소담 시뮬레이터는 **"참고용·세무사 검토 전" 워터마크**. → 세무사법 위험선 회피.

### 4-2. 통합고용세액공제 특화 (SODAM 유일의 진짜 데이터 해자)
- 직원 급여·4대보험·출퇴근 데이터로 **상시근로자 수를 월별 정확 증빙** → **통합고용세액공제(고용증대)** 정밀 산정 + **사후관리 추징 방어**(세무사가 가장 놓치고 추징당하는 항목).
- 환급(기납부세액 반환)이 아니라 **절세(세액 감소, 1인당 수백만원)** → 수수료 여력 큼. 정액 200,000~300,000원 패키지 또는 절세액 10%. 세무사도 추징 리스크↓로 환영.
- **포지셔닝을 "환급 플랫폼"이 아니라 "직원 쓰는 사장님 고용세액공제 특화"로 재정의.**

### 4-3. 반복 세무 보조 (진짜 LTV는 연 1회 환급이 아니라 매월·분기)
- 소담은 급여·4대보험·원천징수 데이터를 **매월 보유** → **부가세 신고 보조·매입세액 누락 점검·원천세 자료 생성**이 데이터 적합성이 높고 **분기/월 반복 매출**(LTV 큼).
- 종소세(연 1회)는 시즌 캠페인, **부가세·원천세·4대보험은 상시 구독 가치**로 PRO/PREMIUM에 내장.

### 4-4. 세무 도입 게이트 (M6 선결 — 순서 중요)
1. **세무사법 유권해석/법률 검토 통과**(어디까지가 SW, 어디부터가 세무대리) — **세무사 계약보다 먼저**.
2. 세무사 파트너 계약 ≥ 1건.
3. 정산·송객 코드 E2E 1건(현재 환급 코드 0%).
4. 환불·책임 소재·약관 문구 확정(법률 = 인간 승인).
> 통과 전 세무 매출 인식 0. 구독만으로 BEP 검증(§3). 세무는 검증된 **업사이드**.

## 5. 메인(위원장) 최종 판정 — 4인 토론 쟁점

| 쟁점 | 입장 | **판정 & 근거** |
|---|---|---|
| 직원 수 과금 | CEO/CFO 채택 vs 사장님 "직원 숨김 유발" | **사장님 채택** — FREE 2·STARTER 5만 게이트, PRO부터 무제한. 차별화는 기능 |
| 4~7명 빈 구간 | 사장님 지적 | **수용** — STARTER 5명 + PRO 무제한으로 구간 해소 |
| FREE 직원 한도 | CFO 1명 vs 사장님 2명 | **사장님 채택** — 1명은 가치 경험 전 페이월 |
| 멀티매장 가격 | +9,900원 vs 사장님 "2호점 밀어냄" | **사장님 일부 채택** — 첫 추가 +4,900원, 3호점부터 9,900원 |
| 환급 정률 모델 | CFO/CEO 15% vs 세무사 "폐기" | **세무사 채택** — 정액 신고 송객 + 고용공제 + 반복세무로 피벗 |
| 세무사법 리스크 | 세무사 "BLOCKER" | **수용** — 유권해석을 M6 최우선 게이트로. SODAM은 세무대리 안 함 |
| PREMIUM 세무사 1:1 | CEO 번들 vs 세무사 "명의대여" | **세무사 채택** — 삭제, "파트너 우선연결(실비 별도)" |
| 환급 최소수수료 역진성 | 사장님 "최소 3만원 바가지" | **무의미화** — 정률 폐기로 해소(정액 패키지) |
| 연납 부담 | CEO 2개월무료 vs 사장님 "선결제 무서움" | **절충** — 연납 + 반년납 + 일시정지 병행 |

## 6. GTM·성장 레버 (v2.0)

1. **전환 트리거(제품 내 자동 펌프)**: ① **주휴수당 발생 경고**(금액 표시, 미지급=체불 공포) ② **명세서 발급 페이월**(월급날) ③ **4대보험 신고 시점**(알바 입·퇴사) ④ 인건비 비율 경고.
2. **직원→사장 바이럴(CAC 0)**: 알바가 명세서·출퇴근으로 SODAM 인지 → 명세서 하단 **"SODAM으로 급여를 정확히 지급합니다" 워터마크** = 무료 광고 + 신규 사장 역추천.
3. **세무사 화이트라벨(B2B2B, 세무사법 안전)**: 세무사가 자기 고객에게 SODAM(급여·증빙 SW)을 깔게 하고 소담은 구독료 — 신고는 세무사가, 소담은 클린 SaaS.
4. **고용세액공제 캠페인**: "직원 쓰면 받는 세금공제, 놓치면 추징" — 데이터 해자 직결 메시지.
5. **가격 A/B 우선순위**: ① STARTER 9,900원 전환·업셀률 ② FREE 2명 vs 3명 ③ 멀티매장 첫 추가 4,900원 탄력성 ④ 정액 신고 패키지 가격대.
