## 5. 잔여 = 인간·외부 액션 (코드 불가)

1. 🔑 **토스 실키 주입 + `mode=live` + `FLYWAY_ENABLED=true`** → 결제 작동([런북](../02-runbook/결제-라이브-전환.md))
2. 🔑 **react-native-webview 설치**(FE 결제창) · FCM/카카오/채널톡 실키
3. ⚖️ **약관·개인정보·전자금융·환불정책 변호사 검토**(법적 문구 = 인간)
4. 🏛 **위치기반서비스 사업 신고**(행정)
5. 🏛 **세무사·노무사 파트너 계약 + 세무사법·노무사법 유권해석**(세무·노무 송객 매출 전제)
6. 🏛 **토스 지급대행 계약**(세무사 payout 자동화)
7. 🚀 **Play Store 등록·심사**, 운영 배포(AWS), 마이그레이션 실행

> ❌ 법적 라인(키로도 불가): 주민번호 저장·4대보험 EDI 대행·세무/노무 대행. 합법 대체(서식·송객)만 구현됨.

## 6. 권장 출시 순서

1. 변호사 약관 검토 완료 → 2. 위치기반 신고 → 3. 토스/FCM 실키 + webview 설치 → 4. 스테이징에서 결제 e2e → 5. Flyway 실행(백업) → 6. Play Store 내부테스트 → 7. 단계 배포.
