# G-1 · G-2 출시 차단 — 인수인계 (코드 완료 / 인간·전문가 잔여)

**작성**: Claude (자율 작업 종료 시점)
**대상**: 대표자 · 변호사 · 행정사
**상태**: 두 블로커의 **소프트웨어 구현 100% 완료**, 비(非)코드 항목만 잔여

> 이 문서는 출시 차단 G-1(위치정보법)·G-2(카카오 동의 우회)의 **남은 작업과 코드 연동 지점**을
> 인계하기 위한 것이다. 코드로 닫을 수 있는 부분은 모두 닫혔고, 잔여는 **법률 문구 확정·행정 신고**뿐이다.

---

## G-1 — 위치정보법 (GPS 출퇴근)

### ✅ 코드 완료
| 항목 | 위치 |
|---|---|
| 위치정보 동의 수집(필수 분리) | `ConsentService.recordConsents` / `POST /api/auth/consents` (`locationInfoAgreed`) |
| 동의/철회 단건 | `ConsentController.recordLocationConsent` / `PUT /api/auth/consents/location?agreed=` |
| **무동의 GPS 출퇴근 차단** | `AttendanceService.assertLocationConsent` — 좌표 수집·검증 전 동의 강제 |
| 동의 이력·버전 보존 | `TermsAgreement`(LOCATION_INFO, `location-v1.0`) |
| FE 위치동의 API | `authApi.setLocationConsent` |
| Manifest 위치권한 | `ACCESS_FINE/COARSE_LOCATION` + `uses-feature`(required=false) 정상 |

### 🔒 잔여 (인간·전문가)
1. **[행정사/대표자] 위치기반서비스사업 신고** (방통위/KISA) — 신고 수리 전 상용 제공 시 형사리스크
2. **[변호사] 위치기반서비스 이용약관 확정** — 골격: [`docs/legal/location-based-service.md`](../legal/location-based-service.md)
3. (권장) 위치정보 이용·제공사실 **확인자료 보존 로그** 구현 — §16② (현재 Attendance 좌표만 저장)

## G-2 — 카카오 가입 약관 동의 우회

### ✅ 코드 완료
| 항목 | 위치 |
|---|---|
| 카카오 가입 시 동의 미수집 마킹 | `KakaoAuthService.saveKakaoUserInfo` (`consentCompleted=false`) |
| 동의 완료 여부 노출 | `UserResponseDto.consentCompleted` (`/api/auth/me`) |
| **동의 강제 진입 게이트** | `AuthNavigator` — `consentCompleted===false` → ConsentScreen 강제 |
| **필수/선택 분리 동의 UI** | `ConsentScreen` + `ConsentBlock`(필수3·선택1, 전체동의, 약관 보기 모달) |
| 동의 기록 | `authApi.recordConsents` → `POST /api/auth/consents` → `ConsentService` |
| 동의 이력·버전 보존 | `TermsAgreement`(TERMS_OF_SERVICE/PRIVACY_POLICY/AGE_14/MARKETING) |
| 필수 동의 검증(BE) | `ConsentService.recordConsents` — 필수 누락 시 거부 |

### 🔒 잔여 (전문가)
1. **[변호사] 약관 본문 확정** → `ConsentBlock.legalTexts` 에 주입 (현재 placeholder)
   - 이용약관 [`terms-of-service.md`](../legal/terms-of-service.md) · 개인정보 [`privacy-policy.md`](../legal/privacy-policy.md) · 마케팅 [`marketing-consent.md`](../legal/marketing-consent.md) — 모두 "초안" 배너 상태
2. **[변호사] 면책조항** — 약관 내 SaaS 책임 한계 조항 (Claude 작성 금지 항목)

> UI 흐름 자체는 완성. 변호사 문구만 `legalTexts` 로 주입하면 즉시 운영 가능.

## 코드 적용 지점 한눈에 (변호사 문구 주입)

```
ConsentBlock.legalTexts = {
  terms:    <docs/legal/terms-of-service.md 확정본>,
  privacy:  <docs/legal/privacy-policy.md 확정본>,
  marketing:<docs/legal/marketing-consent.md 확정본>,
  // 위치기반: GPS 출퇴근 진입 화면에서 location-based-service.md 표시
}
```
→ 이 한 곳만 채우면 G-2 약관 표시가 완결된다. 백엔드·동의수집·이력·강제진입은 이미 동작.
