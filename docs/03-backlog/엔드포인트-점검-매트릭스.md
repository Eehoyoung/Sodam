## 점검 요약
- **수리됨**: 매장 조회/통계 IDOR 2건, 카카오 state, GPS 위치동의 강제, 급여 코어/최저임금 등(노무·법무 커밋 `09c8823` 포함).
- **동작확인**: 권한 어노테이션(@MasterOnly/@EmployeeOrMaster) + 도메인 가드(StoreAccessGuard) 존재, 핵심 도메인 테스트 통과.
- **해당없음**: TestController(dev), Legacy 프록시, Toss 웹훅(서명검증).
- **후속 권장(인계서)**: PersonalUserController 경로 userId 본인 검증 명시화, 원시 엔티티(Store/Attendance) 반환의 DTO 전환(점진).
