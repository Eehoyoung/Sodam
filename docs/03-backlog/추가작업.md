## 3. BE API 커버리지 매트릭스

### 3-1. 구현 완료 (출시 가능)
- 인증: `/api/login`, `/api/join`, `/api/auth/refresh`, `/api/auth/me`, `/kakao/auth/proc`
- 사용자: `/api/user/*`, `/api/users/{id}/purpose`, `/api/user/{id}/convert-to-owner`
- 매장: `/api/stores/registration`, `/api/stores/{id}/location`, `/api/stores/{id}/employees`, CRUD
- 출퇴근: `/api/attendance/check-in`, `/api/attendance/check-out`, `/verify/nfc`, `/verify/location`, `/manual-register`
- 시급: `/api/wages/*` 5종
- 급여: `/api/payroll/calculate`, `/{id}`, `/{id}/status`, `/{id}/details`
- 구독: `/api/billing/plans`, `/me`, `/subscribe`, `/subscribe/free`, `/cancel`, 웹훅
- 알림: `/api/notifications/token` 등록/해제
- 정보허브: 노무/세무/정책/팁/Q&A 5종
- 통계: `/api/store-queries/{id}/stats/today`, `/payroll/month-to-date`
- 휴가: TimeOffController 일부
